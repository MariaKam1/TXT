1) Git - GitBash командная строка (копирование, создание, просмотр, перемещение файлов на серверах без графического интерфейса; работа в GitHub (система контроля версий);
2) DevTools - использование в браузерах инструментов разработчика, позволяющие посмотреть исходный код сайта для отслеживания работы HTML,CSS,JavaScript, и для проверки сетевого трафика, быстродействия сайта и т.д;
3) SQL (базы данных) - использование команд: create, delete, drop, insert into, select, from, where, join;
4) VPN (виртуальная частная сеть) - технология, позоляющая создать безопасное подключение пользователя к сети, организованной между несколькими компьютерам; Варианты использования инструментов;
5) Charles - инструмент мониторинга и перехвата HTTP/HTTPS трафика;
6) Jmeter - использование данного инструмента для проведения нагрузочного тестирования;
7) Postman - инструмент для работы с API (отправка запросов с клиента на сервер и получение ответа с сервера)
