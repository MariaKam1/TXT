Git - GitBash командная строка (копирование, создание, просмотр, перемещение файлов на серверах без графического интерфейса; работа в GitHub (система контроля версий);
DevTools - использование в браузерах инструментов разработчика, позволяющие посмотреть исходный код сайта для отслеживания работы HTML,CSS,JavaScript, и для проверки сетевого трафика, быстродействия сайта и т.д;
SQL (базы данных) - использование команд: create, delete, drop, insert into, select, from, where, join;
VPN (виртуальная частная сеть) - технология, позоляющая создать безопасное подключение пользователя к сети, организованной между несколькими компьютерам; Варианты использования инструментов;
Charles - инструмент мониторинга и перехвата HTTP/HTTPS трафика;
Jmeter - использование данного инструмента для проведения нагрузочного тестирования;
Postman - инструмент для работы с API (отправка запросов с клиента на сервер и получение ответа с сервера)
